# Angular 2 - todo list

Demo angular 2 application

### Usage

1) Clone repository
```
git clone https://github.com/ancor-dev/angular2-todo.git angular2-todo
```
2) Move to `angular2-todo` directory
```
cd angular2-todo
```
3) Install dependencies
```bash
npm install
```
4) Run building...
```bash
npm start
```
5) Go to [http://localhost:3000](http://localhost:3000)
